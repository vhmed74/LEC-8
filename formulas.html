خصائص الأجهزة / Instrument Characteristics

%Inaccuracy — الدقة
%Inaccuracy =
(Max Error / Full Scale) × 100%
Full Scale = Max Input − Min Input
Sensitivity — الحساسية
S = ΔOutput / ΔInput
الوحدة = وحدة الخرج ÷ وحدة الدخل
Unit: (output unit) / (input unit)
First-Order Response — الثابت الزمني
T(t) = T(∞) + [T(0)−T(∞)] · e−t/τ
عند t = τ → تصل إلى 63.2% من القيمة النهائية
Settling Time — وقت الاستقرار
t95% ≈ 3τ    t99% ≈ 5τ
95% من القيمة النهائية عند 3τ
99% عند 5τ

الفوتون وتفاعله / Photon Interactions

Photon Energy — طاقة الفوتون
E = hν = hc / λ
h = 6.626×10⁻³⁴ J·s  |  c = 3×10⁸ m/s
λ بالمتر — E بالجول
L4

البصريات والانكسار / Optics & Refraction

Snell's Law — قانون سنيل
n₁ sin(θ₁) = n₂ sin(θ₂)
θ تُقاس من العمود على السطح
θ measured from the normal
Critical Angle — الزاوية الحرجة
θc = sin⁻¹(n₂ / n₁)
يشترط n₁ > n₂
TIR يحدث عند θ ≥ θc
Reflectance — الانعكاسية (عمودي)
R = [(n₂ − n₁) / (n₂ + n₁)]²
الإرسالية: T = 1 − R (بدون امتصاص)
Transmittance T = 1 − R
Refractive Index — معامل الانكسار
n = c / v
c = سرعة الضوء في الفراغ
v = سرعته في الوسط
L5

الامتصاص والمطيافية / Absorption & Spectroscopy

Beer-Lambert (Exponential)
I = I0 · e−αx
α = معامل الامتصاص (cm⁻¹)
x = طول المسار (cm)
Beer-Lambert (Absorbance)
A = −log₁₀(I / I0) = ε · c · L
ε = Molar absorptivity  |  c = Concentration
L = Path length (cm)
L6

متغيرات الليزر / Laser Variables

Irradiance — كثافة الإشعاع
E = P / A   [W/cm²]
A = مساحة البقعة
A = πr² حيث r = نصف القطر (cm)
Fluence — التدفق الضوئي (الجرعة)
H = E × t = P·t / A   [J/cm²]
t = زمن التعرض (ثانية)
t = exposure time (seconds)
Average Power — متوسط القدرة (نبضي)
Pavg = Epulse × f
f = معدل التكرار (Hz)
Epulse = طاقة النبضة (J)
L7

تفاعل الضوء مع الأنسجة / Light–Tissue Interaction

Reduced Scattering Coeff. — التشتت المخفض
μ's = μs (1 − g)
g = عامل عدم التناسق (0 = كل الاتجاهات، 1 = أمامي)
g = anisotropy factor
Effective Attenuation — التوهين الفعال
μeff = √[3μa(μa + μ's)]
يجمع تأثيري الامتصاص والتشتت
Combines absorption + scattering
Optical Penetration Depth — عمق الاختراق
δ = 1 / μeff   [cm]
عند العمق δ تنخفض الشدة إلى 1/e ≈ 37%
Intensity drops to 37% at depth δ
Intensity at Depth z — الشدة عند العمق
I(z) = I0 · e−μeff·z
z = العمق داخل الأنسجة (cm)
z = depth in tissue (cm)

مبادئ الليزر / Laser Principles

Pulse Energy — طاقة النبضة
Epulse = Ppeak × tpulse
tpulse = مدة النبضة
Ppeak = قدرة الذروة
Frequency Doubling SHG — مضاعفة التردد
λout = λin / 2
مثال: Nd:YAG 1064 nm → 532 nm (أخضر / green)
